This project is a simple shell written in c for a school project.

Shell commands are :

islem : command to indicate a calculation will be made
 
cikar : This command is a parameter to islem command. Subtracts two number given as input. 

topla : This command is a parameter to islem command. Adds  two number to each other  given as input.

tekrar : Takes two parameter, a string and an integer. Shell prints the string as much as the integer.

cat : Shell calls the system command cat.

Multiple commands can be given by using the " | " sign between each command.
